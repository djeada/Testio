# single_input

Runs `program.py` with one input value and checks that the program doubles it.

## Files
- `program.py`: Python program that reads one value.
- `config.json`: Uses `python3` for 1 test with input `3`, expecting output `6`.

## Run
From inside this directory, run:
```bash
testio run config.json
```

## What this demonstrates
- Supplying a single stdin line.
- Validating a simple one-input transformation.
